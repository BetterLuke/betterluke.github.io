2018-01-02

2018-02-05

一部电影没有看完，是《局外人》，身为高能交易员的主人公，从一个职场新人，最后成长到能力逆天的boss man，自然而然的因为贪婪与自信犯下了滔天大错，为08年金融危机加了一把火。看到主人公手速与语速都飞起的片段，我想到了之前和大家一起上线的画面。

“切环了吗？”
“切了，好的，马上全量重启！”
“为什么这么多报警邮件？！”
“哦，是探针没有停”

下午我们也聊了些，我看见她懒洋洋的躺在床上还没来得及迎接小麦色的日出，多想在这个时间一起晨跑，仔细想好已经逝去的上午该如何忙过。

晚上我在的时候，她那边已经是下午，小组貌似正在讨论工作分配，看她一脸迷糊样，该鼓励的鼓励，分析的分析，然后，让她马上放下手机，去加入和他们的讨论。太阳每天升起，落下，每一天的生活都是你的选择，你可以让自己打扮的漂亮的，心态积极的。所以，思佳同学，笑起你的脸儿，make it better :)

2018-02-06

是的，我还是想着没有到的快递，EMS啊拜托，你看咱们的高铁速度，而你呢？

中午，我拿起了4年前买的吉他，在左同学的帮助下，知道了C，DM，AM的和弦的按法，他说，四个万能的和弦你练习熟悉了，就可以弹奏很多的歌曲了。像许巍的歌曲啊等等。

弹吉他，唱歌，一件你向往的事情，搁置起来，却已不知是四年。时间飞速的向前走，还是二十岁的年纪，你还可以，做一些有趣的事情。

我们究竟浪费了多少的热情在自己想象做一件的时候，大多是时候，我们就习惯了提前享受一件还没有做完的事情带来的成就感，这样不好，人脑的奖励机制是既定的，还没有尝到坚持一件事情的苦头，就臆想将来虚无缥缈的甜，所以做事情你总是放弃。而其实，最可悲的是，你还没有尝到真正的半点的甜。我想我就常常如此，还没有学会一个语言，就臆想到自己想要实现的种种，想的太多，思考需要的支持却很少，因为我还没有让自己变为可以允吸这个领域知识的海绵，在没有一定的基础知识下，提前看了好多若不相关的文章，结果自然就是，大多数是石头。

既然出名要趁早，有了做一件事情的小小冲动，freed it.

今天加入了UKU小分队，刚刚讨论组里某某分享的歌曲，大概是让彼此早点休息。:)
