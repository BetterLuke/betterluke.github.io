1. Click

与命令行交互，使用click就可以更多时间啊专注于业务，问不是处理获取参数相关的逻辑。用之前做过一个小工具举例，

需求是这样的：

  • 从用户终端获取两个参数，分别是两个Git仓库的tag
  • 获取一个参数，为用户接受构建结果的邮箱
  • 最好可以交互提示用户是否进行下一步
  • 使用获取的参数构建一个请求

对于上面这个工具的需求，其实很多工作都在命令行获取参数上，使用click实现的代码：

@click.option('--mail',prompt="mail(请输入你名字的拼音以接受邮件)")
@click.confirmation_option(prompt="请再次检查tag！继续？")
